C盘生活垃圾清理指南

2022 / 07 / 20

最近、C盘又开始爆红、介于手 头都很紧、120G 的固态也只能留着回家过年再换(¿)
今天来看看除了网上的通用教程以外、还有什么可以删的地方罢。

非常重要、紧急的Tip：别忘了删除文件后右键回收站、【清空回收站】再看效果！


用到的工具:

搜寻文件神器 Everything https://www.voidtools.com/zh-cn/
统览占用秘宝 SpaceSniffer http://www.uderzo.it/main_products/space_sniffer/download.html

垃圾分类:

Log | 运行日志、这种玩意就是大家8杆子打不着的 程序运行的时候写的日记 (不要闲的没事干去读，因为写的全是流水账)

位置:

程序残留：
| 在下列文件夹中找找有没有你【已卸载的应用的名字】，如果确实不需要保留那些应用数据就 全！部！删！掉！
| C:\Users\你的windows用户名\AppData\Local
| C:\Users\你的windows用户名\AppData\Roaming

| 在 Everything 搜索【"C:\Users" "\pending" .exe】、点击上方【大小】以文件大小排序、这部分是升级安装包、根据程序的图标和路径判断是否可以删除这些

浏览器：
| 浏览器可能是最会产生垃圾的应用之一，如果你正在使用edge，可以用 SpaceSniffer 查看 C:\Users\LSDog\AppData\Local\Microsoft\Edge\UserData\Default、占用还是蛮大的
| C 盘不够用时去浏览器的设置界面清除所有数据（当然这样肯定会掉网站登录）会有不错的效果

腾讯会议：
| 吐槽：腾讯会议给每个会议室都复制了一遍自带的背景图
| 日志 C:\Users\LSDog\AppData\Roaming\Tencent\WeMeet\Global\Logs
| 聊天区图片 C:\Users\LSDog\AppData\Roaming\Tencent\WeMeet\Global\IM\image
| 虚拟背景图 C:\Users\LSDog\AppData\Roaming\Tencent\WeMeet\Global\VirtualBkg

音乐软件：
去听歌软件里的设置找到【清理缓存】清理即可

python 的 pip 缓存：
| 如果你使用 python
| 使用 Win + R 键打开命令行窗口
| 输入 cmd 回车：
| 执行这两行可以查看、清除缓存：
python -m pip cache info python -m pip cache remove
| 示例/执行输出：
C:\Users\LSDog>python -m pip cache info Package index page cache location: c:\users\lsdog\appdata\local\pip\cache\http Package index page cache size: 743.6 MB Number of HTTP files: 360 Wheels location: c:\users\lsdog\appdata\local\pip\cache\wheels Wheels size: 101.2 MB Number of wheels: 10 C:\Users\LSDog>python -m pip cache purge Files removed: 370 C:\Users\LSDog>python -m pip cache info Package index page cache location: c:\users\lsdog\appdata\local\pip\cache\http Package index page cache size: 0 bytes Number of HTTP files: 0 Wheels location: c:\users\lsdog\appdata\local\pip\cache\wheels Wheels size: 0 bytes Number of wheels: 0

杂项：
| PowerToys下载缓存 C:\Users\LSDog\AppData\Local\Microsoft\PowerToys\Updates
| 完美世界html缓存 C:\Users\LSDog\AppData\Roaming\perfectworldarena\Cache
| Epic html缓存 C:\Users\LSDog\AppData\Local\EpicGamesLauncher\Saved\以webcache开头的文件夹
| Steam html缓存1 C:\Users\LSDog\AppData\Local\Steam\htmlcache\Cache
| Steam html缓存2 C:\Users\LSDog\AppData\Local\Steam\htmlcache\Code Cache
| 反正删了没问题的 C:\Windows\ServiceProfiles\LocalService\AppData\Local\Temp | 安装包 C:\Users\LSDog\AppData\Local\Downloaded Installations | 安装包 C:\Users\LSDog\AppData\Roaming\Downloaded Installations
